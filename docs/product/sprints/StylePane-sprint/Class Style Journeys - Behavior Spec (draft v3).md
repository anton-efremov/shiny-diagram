## Vocabulary

- **Direct style** — inline `style Foo …`, one class, unnamed.
- **Named style** — `classDef Name …`, shared, named, applied via `cssClass` / `:::`.

---
## Style properties supported in Shiny:

`fill`, `stroke`, `stroke-width`, `stroke-dasharray`, `color`

## Style StylePane

| Section                    |          | Behavior                                                                                                                                                                           |
| -------------------------- | -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Header** (swatch + name) | Elements | swatch box of `S` + name **"S"** in swatch header                                                                                                                                  |
|                            | Effect   | click name → <br>- edits stored locally until submitted<br>- on "Enter" or click outside - change submitted (with collision prevention mechanism)<br>- on "Esc" - change discarded |
| **Style controls**         | Elements | `S`'s values                                                                                                                                                                       |
|                            | Effect   | edit → update `S` for all wearers                                                                                                                                                  |
| **Delete style**           | Elements | **[Delete style]**                                                                                                                                                                 |
|                            | Effect   | removes class definition and all assignements<br>moves to unselected state                                                                                                         |
| **Exit**                   | Elements | (implicit — click outside that changes selection)                                                                                                                                  |
|                            | Effect   | Changes Style StylePane view to whatever is selected                                                                                                                               |
