# Sprint 1.1 — Class Box

## Goal

Deliver a complete class-box journey as an editor object: create, select, move, resize, style, duplicate, and delete boxes, including multi-selection and group movement.

Exclude member operations, connector behavior, undo/redo, protocol changes, and view-state refactoring.

## Tasks

1. **Selection** — Select a box by clicking it and clear or replace selection predictably.
2. **Move** — Move a box from its available drag surfaces and persist its position.
3. **Resize** — Resize a selected box from eight usable handles and persist its rectangle.
4. **Visual states** — Make normal, hovered, selected, moving, and resizing states visually clear.
<-------------done-------------->
5. **Delete** — Delete selected boxes through the box-level UI and remove them from source.
6. **Add** — Add a new class box from the tool pane with a unique name and initial placement.
7. **Duplicate** — Duplicate the selected box from the style pane, including its box-level properties.
8. **Multi-select** — Select and deselect multiple boxes using standard editor interactions.
9. **Group move** — Move all selected boxes together and persist every resulting position.
10. **Style controls** — Edit box-level visual properties from the style pane.
